# Anis Natasha — Personal Portfolio Blog

> A clean, responsive personal portfolio and blog website built with pure HTML, CSS, and JavaScript as part of the CSD 34203 Individual Project (UniSZA).

---

## 📌 Project Description

This is a multi-page personal portfolio and blog website that showcases my journey as a Software Development student at Universiti Sultan Zainal Abidin (UniSZA). The site covers my projects, coding experience, technical blog posts, and contact information. It is fully responsive and features modern animations.

---

## ✨ Features

- **Home Page** — Hero section with typing animation, project showcase, and blog preview
- **About Page** — Personal bio, animated skill progress bars, education timeline, and hobbies
- **Blog Page** — 3 sample posts with category filter and featured post highlight
- **Contact Page** — Contact form with input validation and success feedback
- **Mobile Responsive** — Hamburger menu and adaptive layouts for all screen sizes
- **Smooth Animations** — Typing effect, floating avatar, fade-in on scroll, animated skill bars

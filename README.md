# Ahmad Ridhwan — Personal Blog

> A clean, responsive personal blog built with pure HTML, CSS, and JavaScript as part of the CSD 34203 Individual Project (UniSZA).

![Blog Screenshot](screenshots/preview.png)

---

## 📌 Project Description

This is a multi-page personal blog site that showcases my journey as a computing student — covering web development projects, learning tips, and tech reflections. The site is fully responsive and includes a dark mode toggle.

---

## ✨ Features

- **Home Page** — Hero section with intro, stats, and featured blog posts
- **About Page** — Bio, skills progress bars, education timeline, and social links
- **Blog Page** — 3 sample posts with category filtering and full single-post view
- **Contact Page** — Contact form with validation and success feedback
- **Dark Mode** — Toggle between light/dark themes (preference saved in localStorage)
- **Mobile Responsive** — Hamburger menu and adaptive layouts for all screen sizes
- **Smooth Navigation** — Single-page app feel with no page reloads

---

## 🛠 Technologies Used

| Technology | Purpose |

## 🚀 How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahmadridhwan/personal-blog.git
   cd personal-blog
   ```

2. **Open in browser:**
   ```bash
   # Simply open index.html in any modern browser
   open index.html
   # OR use VS Code Live Server extension
   ```

No build tools, no dependencies, no installation required.
